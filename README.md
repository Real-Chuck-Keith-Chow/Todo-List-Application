# To-Do List Application

A command-line To-Do List application written in C.

This program allows users to manage tasks through a simple menu interface. Tasks can be added, deleted, and displayed, with support for assigning deadlines and validating dates.

## Features

* Add tasks
* Delete tasks
* Display tasks
* Store task deadlines
* Validate dates

## Technologies

* C
* Standard C Library

## Build

```bash
gcc todo.c -o todo
```

## Run

```bash
./todo
```
